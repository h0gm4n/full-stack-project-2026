# full-stack-project-2026

## Project description

Simple app for managing e.g. reservations and messaging in an apartment building context. 
<br>Current ideas include:</br>
<ul>
<li>Creation of channel for estate (for admin / property manager)</li>
<li>Ability to join a channel for estate (for resident)</li>
<li>Ability to view and edit personal information (for resident)</li>
<li>Info board</li>
<li>Reservation system for e.g.
    <ul>
        <li>Laundry room</li>
        <li>Sauna</li>
        <li>Activity room</li>
    </ul>
</li>
<li>Parking spot rental with queue</li>
<li>Discussion forum for residents</li>
</ul>
